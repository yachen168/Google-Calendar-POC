# Google Calendar
**demo page**: https://peaceful-elion-1b2bf3.netlify.app
## 目標
1. 在使用者使用 google 登入後，獲取使用者 google calendar 資料
2. 將獲取的資料繪製於 calendar 套件
3. 在繪製的 calendar 上新增 event

主要使用的 google calendar api: 
- get events
- create events
